### 5.2 Dotation Jeunes Agriculteurs (DJA)

| Critère | Condition |
|---------|-----------|
| **Âge** | 18-40 ans à la date d'installation |
| **Capacité professionnelle** | Diplôme niveau 4 agricole (Bac Pro, BPREA) + PPP |
| **Plan d'entreprise** | Viabilité économique démontrée |
| **Première installation** | Qualité de chef d'exploitation pour la 1ère fois |
| **Assujettissement social** | MSA comme non-salarié |

#### Montants DJA (Cadre 2023)

| Zone | Montant de base | Modulations possibles |
|------|-----------------|----------------------|
| Plaine | 10 000-15 000 € | +5 000 € hors cadre familial |
| Défavorisée simple | 15 000-22 000 € | +5 000 € agroécologie |
| Montagne / Haute montagne | 22 000-36 000 € | +3 000 € projet innovant |

## 7. Statuts et Structures d'Exploitation

### 7.1 Formes Juridiques

| Forme | % exploitations | Caractéristiques |
|-------|-----------------|------------------|
| **Exploitation individuelle** | 55% | Personne physique, patrimoine confondu |
| **EARL** | 25% | 1-10 associés, responsabilité limitée |
| **GAEC** | 12% | Travail en commun, transparence |
| **SCEA** | 5% | Souplesse, responsabilité illimitée |
| **Autres (SA, SARL...)** | 3% | Cas particuliers |

### 7.2 GAEC — Groupement Agricole d'Exploitation en Commun

| Élément | Règle |
|---------|-------|
| **Objet** | Travail en commun |
| **Associés** | 2-10 personnes physiques |
| **Transparence** | Chaque associé = 1 exploitant (aides PAC) |
| **Responsabilité** | Illimitée mais subsidiaire |
| **Capital** | Variable |
| **Agrément** | Préfet de département |

### 7.3 EARL — Exploitation Agricole à Responsabilité Limitée

| Élément | Règle |
|---------|-------|
| **Objet** | Activité agricole |
| **Associés** | 1-10 (dont 50% exploitants minimum) |
| **Responsabilité** | Limitée aux apports |
| **Capital** | 7 500 € minimum |
| **Gérance** | Associé exploitant obligatoirement |

### 7.4 GFA — Groupement Foncier Agricole

| Type | Objet | Spécificités |
|------|-------|--------------|
| **GFA familial** | Portage foncier familial | Exonération droits mutation |
| **GFA-bailleur** | Location à exploitant | Fiscal avantageux |
| **GFA-investisseur** | Placement | Depuis 2014, régime spécifique |

### 7.5 GIEE — Groupement d'Intérêt Économique et Environnemental

| Élément | Contenu |
|---------|---------|
| **Création** | Loi d'avenir 2014 |
| **Objet** | Projet collectif agroécologique |
| **Forme** | Association, coopérative, GIE... |
| **Reconnaissance** | Préfet de région |
| **Avantages** | Priorité aides, majoration MAEC |
| **Nombre (2024)** | ~500 GIEE reconnus |
