## 2. Les Trois Piliers et Leurs Interactions

### 2.1 Pilier EAU — Missions et Interfaces

| Mission ANE | Interface Agriculture | Interface Alimentation |
|-------------|----------------------|------------------------|
| Gestion ressource | Quotas irrigation, REUT | Besoins IAA |
| Qualité eau | Pollutions diffuses agricoles | Normes eau potable |
| Infrastructures | Réseaux irrigation | Approvisionnement IAA |
| Gouvernance | Comités de bassin | PAT, CTA |

### 2.2 Pilier AGRICULTURE — Missions et Interfaces

| Mission CNA | Interface Eau | Interface Alimentation |
|-------------|---------------|------------------------|
| Production | Besoins en eau | Approvisionnement local |
| Transition agroécologique | Qualité eau | Qualité produits |
| Installation | Accès ressource eau | Débouchés locaux |
| Filières | REUT agricole | Transformation, circuits courts |

### 2.3 Pilier ALIMENTATION (CTA) — Missions et Interfaces

| Mission CTA | Interface Eau | Interface Agriculture |
|-------------|---------------|----------------------|
| Accès alimentation | Qualité eau consommation | Production locale |
| Circuits courts | Logistique | Producteurs du territoire |
| Restauration collective | Normes sanitaires | Approvisionnement |
| Résilience territoriale | Sécurité hydrique | Sécurité alimentaire |

### 2.4 Matrice des Interactions

| | EAU (ANE) | AGRICULTURE (CNA) | ALIMENTATION (CTA) |
|---|-----------|-------------------|-------------------|
| **EAU (ANE)** | — | REUT, irrigation, pollutions | IAA, eau potable |
| **AGRICULTURE (CNA)** | Besoins, pratiques | — | Production, qualité |
| **ALIMENTATION (CTA)** | Consommation, empreinte | Approvisionnement | — |

## 6. Cas d'Usage et Scénarios

### 6.1 Cas 1 : Territoire en Stress Hydrique

**Contexte** : Bassin versant méditerranéen, déficit chronique, conflits d'usage.

| Étape | Action | Pilote | Résultat |
|-------|--------|--------|----------|
| 1 | PTGE (diagnostic partagé) | Agence eau | Carte des tensions |
| 2 | Création CTA | Collectivité | Instance de dialogue |
| 3 | Étude REUT | ANE + CTA | Potentiel 5 Mm³/an |
| 4 | Plan sobriété agricole | CNA + Chambre | -20% prélèvements |
| 5 | Station REUT | Syndicat + Agence | 3 Mm³/an disponibles |
| 6 | Contrats REUT agriculteurs | CTA | 50 exploitations irriguées |
| 7 | Circuits courts légumes | CTA | Cantine scolaire = 100% local |

**Bilan** : Déficit comblé, conflits apaisés, alimentation locale.

### 6.2 Cas 2 : Territoire Pollué (Nitrates)

**Contexte** : Bassin versant breton, captage dégradé, algues vertes.

| Étape | Action | Pilote | Résultat |
|-------|--------|--------|----------|
| 1 | Diagnostic qualité | Agence eau | Captage >50 mg/L nitrates |
| 2 | Périmètre protection | DDT | Zone prioritaire 5 000 ha |
| 3 | Programme MAEC | DRAAF + Agence | 80% SAU sous contrat |
| 4 | Accompagnement transition | CTA + CIVAM | Conversion 30 fermes bio |
| 5 | Valorisation "eau propre" | CTA | Label territorial |
| 6 | Circuits courts bio | CTA | Magasins, cantines |
| 7 | Suivi qualité | ANE | Nitrates <40 mg/L en 5 ans |

**Bilan** : Qualité eau restaurée, filière bio rentable, image territoire.

### 6.3 Cas 3 : Installation Jeune Agriculteur

**Contexte** : JA maraîcher, projet péri-urbain, besoin irrigation.

| Étape | Action | Pilote | Résultat |
|-------|--------|--------|----------|
| 1 | Accueil PAI | Chambre agriculture | Projet validé |
| 2 | Recherche foncier | SAFER + CTA | 5 ha disponibles |
| 3 | Étude accès eau | Syndicat eau | Raccordement REUT possible |
| 4 | Montage financement | Banque + Région | DJA + prêt bonifié |
| 5 | Contrat REUT | CTA (facilitation) | 10 000 m³/an garanti |
| 6 | Débouchés | CTA | Contrat cantine + AMAP |
| 7 | Installation | JA | Exploitation viable J+1 |

**Bilan** : JA installé, accès eau sécurisé, débouchés garantis.

### 6.4 Cas 4 : Crise Sécheresse

**Contexte** : Été caniculaire, restrictions d'eau, tensions sociales.

| Phase | Action | Pilote | Coordination |
|-------|--------|--------|--------------|
| **Anticipation** | | | |
| J-60 | Alerte précoce ANE | ANE | → CNA, CTA |
| J-45 | Réunion CTA élargie | CTA | Tous acteurs |
| J-30 | Plan de délestage | Préfet + ANE | Priorisation usages |
| **Crise** | | | |
| J | Arrêté restriction | Préfet | Concertation CTA |
| J+1 | Communication agriculteurs | Chambre + CTA | SMS, réunions |
| J+7 | Ajustements | CTA | Retour terrain |
| **Post-crise** | | | |
| J+60 | Bilan | ANE + CNA + CTA | Leçons apprises |
| J+90 | Plan résilience | CTA | Investissements REUT |

**Bilan** : Crise gérée collectivement, anticipation renforcée.
